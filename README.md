# 🌊 PresUnivGo — Elite Professional Networking for President University

**PresUnivGo** is a high-fidelity, cross-platform professional networking solution engineered specifically for the **President University** ecosystem. This project implements modern mobile development paradigms, including **Clean Architecture**, **Reactive State Management**, and **AI-Driven Personalization**.

🚀 **Live Deployment:** [PresUnivGo Web App](https://puconnect-9e8fb.web.app)

---

## 💎 Key Highlights

-   **Zero-Storage Architecture**: Engineered for maximum cost-efficiency by eliminating all paid cloud storage dependencies.
-   **Reactive Profile Ecosystem**: Real-time Firestore synchronization ensures profile updates (Name, Jobs, Experience) reflect instantly.
*   **Radiant Design System**: A bespoke "Radiant Magenta & Pristine White" palette, optimized for professional aesthetics and elite visual clarity.
*   **Main Feed Integration**: Centralized campus activity hub for real-time engagement with likes and comments.

---

## 🎨 Radiant Design System

-   **Glassmorphism & Depth**: Utilizes `GlassContainer` widgets with high blur (15-20) and low opacity (0.7-0.8) for a premium feel.
-   **Wave UI Design**: Custom `Bezier` curves for fluid onboarding and authentication flows.
-   **Micro-Animations**: Staggered entrance effects and interactive typing indicators powered by `flutter_animate`.
-   **Elite Typography**: Modern variable fonts (Outfit & Inter) for maximum readability.

---

## 🧠 AI Career Mentor Suite (Powered by Gemini)

PresUnivGo integrates an AI Assistant suite to bridge the gap between education and industry:

-   **AI Career Roadmap**: Milestone-based roadmaps tailored to specific academic majors.
-   **AI CV Analysis**: Structural review and actionable feedback for student resumes.
-   **AI Cover Letter Builder**: Tailored professional letters based on student experience.
-   **AI Smart Post Suggestions**: content ideation based on academic history and profile.
-   **MentorBot**: Real-time guidance for career and campus-related queries.

---

## 🏗️ Technical Architecture

### ⚡ Real-Time Ecosystem
-   **Instant Search**: High-performance prefix search with a 300ms debounce.
-   **Networking Engine**: Real-time connection management with instant Firestore synchronization.
-   **Career Analytics**: Dynamic dashboard featuring profile strength metrics and activity graphs.

### 🧩 Reactive State Management
-   **Riverpod Implementation**: Standard `FutureProvider` and `StreamProvider` families for robust, scalable state handling.
-   **Real-Time Streams**: Continuous data synchronization for Chat, Activity Feed, and Network suggestions.

---

## 🛠️ Tech Stack & Dependencies

-   **Framework**: Flutter 3.24 (Channel Stable)
-   **Backend**: 
    -   **Firebase Auth**: Secure RBAC and OAuth2 integration.
    -   **Cloud Firestore**: Real-time NoSQL primary database.
-   **State Management**: `flutter_riverpod`
-   **Navigation**: `go_router`
-   **Visuals**: `flutter_animate`, `shimmer`, `google_fonts`.
